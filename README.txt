Инструкция для запуска:
1) Скачать cmake
2) Скачать "исходники"
3) Распокавать zip-файл в удобное место на компьюторе
4) Открываем консоль, вводим "cmake" и указываем путь к папке, где распакована папка с игрой
5) С помощью Visual Studio открыть "startt.vcxproj"
6) Выбрать конфигурацию "release" в Visual Studio
7) Нажать в обозревателе решений правой кнопкой мыши на "startt"
8) Нажать кнопку "Собрать"
9) В папке с игрой зайти в папку "release" 
10) Запустить exe-файл
11) Наслаждайтесь игрой!
